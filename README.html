LF trimmer

英文整形・翻訳アプリ

テキストボックスに入力した英文から 改行文字(LF)やハイフネーションを自動的に削除した後、 文章をAPI経由で

  • Google翻訳
  • DeelpL翻訳

のいずれかの翻訳サービスに掛け、結果を出力させる。

手順

  1. 左側のテキストボックスに翻訳したい文章を入力.

  2. 入力した文章からLFやハイフネーションを取り除き、GoogleでGoogle翻訳、 DeepLでDeepL翻訳に掛ける。なお、DeepL翻訳を用いる場合、予めDeepL API用のアカウントを 作り、./module/config.iniのAPI_keyの項に専用のAPIキーを、free_or_proにアカウント種別を入力しておく。

  3. 結果を出力。

その他

  • ./module/figureの各種画像は同名のファイルで差し替え可能.
